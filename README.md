# 📊 Student Marks Analyzer

A command-line based Student Management and Academic Performance Analysis System developed using **Python**, **NumPy**, and **JSON**.

This project allows users to manage student records, perform statistical analysis on marks, generate detailed reports, and store data permanently using JSON files. It demonstrates the practical use of NumPy for numerical computations and Python for building real-world CRUD applications.

---

## 🚀 Features

### Student Management

* Add new student records
* Display all student records
* Search students by ID
* Update student marks
* Delete student records

### Academic Reports

* Calculate total marks
* Calculate average marks
* Generate student report cards
* Calculate percentage
* Assign grades based on performance
* Determine pass/fail status

### Performance Analysis

* Find the class topper
* Calculate class average
* Display highest and lowest class scores
* Perform subject-wise analysis
* Display subject-wise averages
* Display highest and lowest marks for each subject

### Data Persistence

* Save student records using JSON
* Load existing records automatically at startup
* Maintain data between program executions
